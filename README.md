# 🎯 Career Recommendation System

## 📌 Overview
This project is a machine learning based Career Recommendation System that helps users choose suitable career options based on their academic performance and other inputs.

It uses a trained model to predict the top 3 career choices and displays them through a Flask web application.

---

## 🚀 Features
- Predicts top 3 career options
- Uses Machine Learning (Scikit-learn)
- Web interface using Flask
- Easy to use and interactive

---

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- HTML/CSS
